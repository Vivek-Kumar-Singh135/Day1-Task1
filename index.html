What's the Difference?  
Create the Structure
HTML * controls the layout of the content
* provides structure for the web page design
* the fundamental building block of any webpage
Hypertext Markup Language
Stylize the website
CSS * Applies style to the webpage element
* Targets various screen sizes to make web pages responsive
* Primarily handles the "look and feel" of a webpage
Cascading style sheet
Increase interactivity
Javascript * Adds interactivity to a webpage
* Handles complex functions and features
* programmatic code which enhances functionality
